# Racy

- Parcel
- Helmet Async
- Styled-Components
- Express
- Graphql Apollo Client & Server

`npm install racy`

- `racy dev` - dev
- `racy build` - build
- `racy export` - export to static HTML
- `racy serve` - serve a build
- `racy start` - build + serve

BUILDDIR
PUBLICDIR
